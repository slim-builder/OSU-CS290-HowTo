How-To Guide for World Bank API Indicator Queries

Written By Steven Lim

Introduction

The World Bank provides several APIs to allow access to its datasets. One of those APIs involve Indicator queries which allows the API user to retrieve various pieces of World Bank data.

For example, the indicator called "GDP (current US$)" represents the Gross Domestic Product of a country in US dollars. The API user can call the API with a query specifying the GDP indicator, a country such as the United States, and a year such as 2010 to obtain the 2010 United States GDP in units of US dollars.
